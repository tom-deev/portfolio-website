Expense Tracker

  • Add, view, and delete expense entries

Network Packet Analyzer

A low-level network tool built with Python and Scapy to capture and inspect live network packets. Designed to help understand how data travels across a network — protocol by protocol, packet by packet.

Key Features

  • Real-time packet capture on selected network interface
  • Supports TCP, UDP, ICMP, and ARP protocols
  • Displays source/destination IP and port info
  • Packet count and protocol distribution summary
  • Filter packets by protocol type
