#  Personal Portfolio Website

This is my personal portfolio website built using HTML, CSS, and JavaScript.
It showcases my projects, skills, and background as a developer in a clean and modern UI.

---

##  About the Project

I created this portfolio to present my work in a structured and visually appealing way.
The focus of this project is strong UI/UX design, responsiveness, and clean layout using core web technologies without any frameworks.

---

##  Tech Stack

* HTML5
* CSS3 (Flexbox, Grid, Animations)
* JavaScript (DOM manipulation, interactions)
